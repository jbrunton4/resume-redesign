Discord Translator Bot

Using the Discord API and the Google Translate Ajax API, I developed a Discord bot to seamlessly translate between over 100 languages.

The bot separates message content into an ISO code and a message, then passes it to the Ajax API. It copies the user's display name and profile picture, then sends this content to a Discord webhook, giving the impression that the user themself sent the translated message.

So long as a user knows the ISO code for a language, they can easily translate into it. The program automatically detects the source language, so there is no need for configuration or extra arguments.

The bot facilitates interlingual communication in environments where it would otherwise be impractical or even impossible.
